# 🎭 Project 14: Persona Prompting Experiment

## What it does
Demonstrates how giving LLM different personas completely
changes vocabulary, tone, and examples in responses.

## Personas Tested
- 🏴‍☠️ Pirate from 1700s
- 👶 5 year old child
- 👨‍🏫 Harvard CS Professor
- 🎙️ Sports Commentator
- 👨‍🍳 5-star Restaurant Chef

## Key Finding
Same question → 5 completely different explanations
Persona affects vocabulary, complexity and examples used.

## Real World Use
- Customer service → friendly persona
- Legal assistant → formal persona
- Kids education → simple fun persona

## Tech Used
- Python, Groq API, LLaMA 3.1
