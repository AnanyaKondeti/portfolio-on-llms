# 🔍 Project 13: LLM Code Reviewer

## What it does
Automatically reviews code for bugs, security vulnerabilities,
performance issues and readability improvements.

## Review Categories
- 🐛 Bugs: Division by zero, null checks
- 🔒 Security: SQL injection detection
- ⚡ Performance: O(n³) → O(n) optimization
- 📖 Readability: Style improvements

## Key Results
- Caught SQL injection vulnerability
- Fixed division by zero bug
- Optimized O(n³) algorithm to O(n)

## Tech Used
- Python, Groq API, LLaMA 3.1
