# 🎯 Project 10: Resume Tailor Bot

## What it does
Rewrites resume bullet points to match specific job descriptions
using LLM-powered language optimization.

## Key Features
- Preserves original facts and experiences
- Injects job-relevant keywords automatically
- Makes existing experience sound relevant to new roles

## Real Use Case
Used to tailor software engineering experience toward
prompt engineering roles at MNCs.

## Tech Used
- Python, Groq API, LLaMA 3.1
