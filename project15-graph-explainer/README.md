# 📊 Project 15: Graph Problem Explainer Bot

## What it does
Explains any graph or computer science algorithm with
pseudocode, complexity analysis and real world examples.

## Algorithms Covered
- Ford-Fulkerson Maximum Flow
- Dijkstra's Shortest Path
- Binary Search

## Output Format
- Overview of algorithm
- Step by step explanation
- Clean pseudocode
- Time & Space complexity
- Real world example

## Connection to Real Work
Directly mirrors Advanced Algorithms coursework at OSU
where Ford-Fulkerson and dynamic programming were implemented.

## Tech Used
- Python, Groq API, LLaMA 3.1
