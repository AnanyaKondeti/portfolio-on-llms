# Graph Problem Explainer Bot 🤖
==========================

## What it does
----------------

The Graph Problem Explainer Bot is an AI-powered tool designed to explain graph algorithms in a clear and concise manner. Using a combination of LLaMA 3.1 for natural language processing and Groq API for performance-critical operations, this bot generates pseudocode and complexity analysis for various graph algorithms. The bot also includes real-world examples and explanations to make graph concepts more accessible to developers and learners alike.

## Key Features
----------------

* Generates pseudocode for common graph algorithms such as Depth-First Search (DFS), Breadth-First Search (BFS), Dijkstra's algorithm, Bellman-Ford algorithm, and more
* Performs complexity analysis for each algorithm to help users understand time and space complexity
* Includes real-world examples to illustrate the use of graph algorithms in various applications
* Supports both directed and undirected graphs

## How to Run
------------

To run the Graph Problem Explainer Bot, follow these steps:

1. Clone the repository using `git clone https://github.com/your-username/graph-problem-explainer-bot.git`
2. Install the required dependencies using `pip install -r requirements.txt`
3. Run the bot using `python src/main.py`
4. Use the web interface to generate pseudocode, complexity analysis, and examples for graph algorithms

## Tech Stack
-------------

* **Programming Language**: Python
* **AI Model**: LLaMA 3.1 for natural language processing
* **API**: Groq API for performance-critical operations
* **Web Framework**: Flask for web development
* **Database**: SQLite for storing algorithm metadata
