**Persona Prompting Experiment** 🤖
==========================

## What it does

The Persona Prompting Experiment is a research project that investigates how different personas impact the response style of Large Language Models (LLMs). This project leverages the Groq API and LLaMA 3.1 to test the effects of various personas on the generated text. The ultimate goal of this experiment is to gain insights into the capabilities and limitations of LLMs in generating human-like responses.

## Key Features

* **5 Different Personas**: This project includes five distinct personas, each with a unique tone, language, and syntax.
* **Vocabulary Analysis**: The experiment includes a vocabulary analysis module to track the use of specific words and phrases across different personas.
* **Tone Comparison**: A tone comparison feature allows for a detailed examination of the tone and sentiment of the generated text across various personas.

## How to Run

### Prerequisites

* python 3.8 or later
* groq-api client library (install using pip: `pip install groq-api`)
* lllama3.1 model (download and install from the Groq API documentation)

### Running the Experiment

1. Clone the repository: `git clone https://github.com/username/persona-prompting-experiment.git`
2. Install the required libraries: `pip install groq-api`
3. Download and install the lllama3.1 model
4. Configure the Groq API settings in the `config.py` file
5. Run the experiment: `python experiment.py`

## Tech Stack

* **Python**: The primary programming language used to develop this project.
* **Groq API**: A cloud-based AI platform that provides access to the LLaMA 3.1 model.
* **LLaMA 3.1**: A large language model used for generating text.

## What I learned

Through this project, I gained valuable insights into the capabilities and limitations of LLMs in generating human-like responses. The experiment revealed how different personas can impact the tone, language, and syntax of the generated text.
