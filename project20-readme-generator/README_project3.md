## What it does
----------------

Chain of Thought Prompting demonstrates the capability of generating step-by-step reasoning using Chain of Thought (CoT) prompting. This project showcases the potential of CoT in solving math problems and provides a comparison with traditional prompting methods.

## Key Features
----------------

* **Math Problem Solving**: Leverages the power of CoT to solve complex math problems, providing a clear and concise step-by-step reasoning.
* **Step-by-Step Reasoning**: Demonstrates the ability of CoT to generate a logical and coherent solution, allowing users to understand the problem-solving process.
* **Comparison with Normal Prompting**: Provides a comparison with traditional prompting methods, highlighting the strengths and weaknesses of CoT.

## How to Run
---------------

To run this project, follow these steps:

1. Install the required dependencies by running `pip install -r requirements.txt`.
2. Set up the Groq API and LLaMA 3.1 as described in the configuration file (`config.yaml`).
3. Run the CoT prompting script using `python cot_prompting.py`.

## Tech Stack
------------

* **Python**: Used as the primary programming language.
* **Groq API**: Utilized for its advanced language understanding capabilities.
* **LLaMA 3.1**: Adopted as the chosen large language model.
