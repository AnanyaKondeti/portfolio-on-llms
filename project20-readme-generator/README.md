# 📝 Project 20: Portfolio README Generator

## What it does
Automatically generates professional GitHub README files
for all 20 portfolio projects using LLM in one run.

## Key Features
- Generates READMEs for all 20 projects in one run
- Professional formatting for each project
- Recruiter-ready descriptions
- Saves each README as separate markdown file

## How to Run
1. Add your Groq API key to `.env`
2. Run `python main.py`
3. Find generated READMEs in the folder

## Tech Stack
- Python, Groq API, LLaMA 3.1

## What I learned
- How to use LLMs for content generation at scale
- Batch processing with LLM APIs
- This is the capstone project combining all 20 projects!
