# LLM Prompt Tester 🧠

**A Python tool for testing and evaluating different prompt styles across Large Language Models**

## What it does
The LLM Prompt Tester is a Python-based tool designed to facilitate the evaluation of various prompt styles across multiple Large Language Models (LLMs). This project utilizes the Groq API and integrates with LLaMA 3.1, a state-of-the-art LLM. By leveraging these technologies, the LLM Prompt Tester provides an efficient way to compare and analyze the performance of distinct prompt styles across different models.

## Key Features

* **Multiple prompt styles**: Supports a variety of prompt styles, enabling users to test and compare their effectiveness across LLM models
* **Response comparison**: Utilizes response analysis to evaluate the quality and relevance of model output for each prompt style
* **API integration**: Leverages the Groq API for seamless interaction with LLaMA 3.1 and other supported LLM models

## How to Run

### Prerequisites
- Install Python 3.8 or later (preferably using a virtual environment, e.g., `python -m venv`)
- Install required libraries via pip: `pip install groq-api`

### Steps
1. Clone the repository: `git clone https://github.com/tm-your-username/LLM-Prompt-Tester.git`
2. Navigate into the project directory: `cd LLM-Prompt-Tester`
3. Run the script using Python: `python main.py`
4. Follow in-app instructions for configuring model parameters and selecting prompt styles
