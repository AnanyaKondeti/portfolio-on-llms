**🤖 ReAct Agent with Tools**
=====================================

**What it does**
----------------

ReAct Agent with Tools is an advanced Large Language Model (LLM) agent designed to reason and employ tools to provide insightful answers to user inquiries. By leveraging the power of natural language processing and specialized tools, ReAct Agent aims to revolutionize the way we interact with artificial intelligence.

**Key Features**
----------------

* **Calculator**: Perform arithmetic operations and solve mathematical problems with ease
* **Weather**: Retrieve current weather conditions and forecasts with just a few words
* **Dictionary**: Define words and explore their meanings, synonyms, and antonyms
* **Reasoning Loop**: Engage in logical reasoning and draw informed conclusions based on provided input

**How to Run**
--------------

### Prerequisites

* Python 3.8+
* Groq API (configured and authenticated)
* LLaMA 3.1 model

### Installation

1. Clone this repository: `git clone https://github.com/username/ReAct-Agent-with-Tools.git`
2. Install required dependencies: `pip install -r requirements.txt`
3. Configure Groq API credentials: `groq config set --api-key YOUR_API_KEY`
4. Run the agent: `python agent.py`

### Example Usage

* Ask a question: `What is the weather like in New York today?`
* Use calculator: `What is 2 + 2?`
* Define a word: `What does 'synchronous' mean?`
