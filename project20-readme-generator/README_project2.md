## Project Overview
This project compares the accuracy of few-shot vs zero-shot prompting on sentiment analysis using the Groq API and LLaMA 3.1.

## Key Features
* Automated benchmarking for few-shot and zero-shot prompting
* Accuracy scoring to compare model performance
* Tricky test cases to evaluate model robustness

### Tech Stack
- **Programming Language:** Python 3.9+
- **API:** Groq API for model deployment and inference
- **Model:** LLaMA 3.1 for language understanding and generation

### How to Run
1. Clone the repository using `git clone`
2. Install required dependencies using `pip install -r requirements.txt`
3. Set up the Groq API credentials and LLaMA 3.1 model details in `config.py`
4. Run the benchmarking script using `python benchmark.py`
