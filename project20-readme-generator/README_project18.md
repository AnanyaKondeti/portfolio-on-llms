# Multimodal Prompting Demo ⚖️
## Description
The Multimodal Prompting Demo is a cutting-edge application that analyzes images with text questions using the power of vision Language Models (LLMs). This innovative solution empowers users to harness the capabilities of LLaMA 4 Scout Vision, unlocking the doors to visual question-answering and image analysis.

## Key Features
* **Image Analysis**: Leverage the power of LLaMA 4 Scout Vision to analyze images and unlock insights hidden within.
* **Base64 Encoding**: Seamlessly integrate with web applications by encoding image analysis results in base64 format.
* **Visual Q&A**: Engage with images and answer questions using the power of vision LLMs.

## How to Run

To set up the Multimodal Prompting Demo, follow these steps:

1. Clone the repository using the following command:

```bash
git clone https://github.com/your-username/Multimodal-Prompting-Demo.git
```

2. Install the required dependencies by running the following command:

```bash
pip install -r requirements.txt
```

3. Replace `YOUR_API_KEY` in the `config.py` file with your actual Groq API key.

4. Run the application using the following command:

```bash
python main.py
```

## Tech Stack

* **Programming Language**: Python

* **LLM**: LLaMA 4 Scout Vision
