**HCI Usability Analyzer**
📊
==========================

A Python-based usability analysis tool to evaluate applications against Nielsen's 10 usability heuristics.

## What it does

The HCI Usability Analyzer is designed to assess the usability of applications by scoring them against Nielsen's 10 usability heuristics. The tool utilizes machine learning capabilities through the LLaMA 3.1 model to detect potential usability issues and provide improvement suggestions. By leveraging the Groq API, this analysis becomes more efficient and precise.

## Key Features

* Scores applications against Nielsen's 10 usability heuristics
* Detects potential usability issues
* Provides improvement suggestions
* Uses machine learning capabilities to enhance accuracy

## How to Run

1. Clone the repository using `git clone https://github.com/username/HCI-Usability-Analyzer.git`
2. Install the required dependencies using `pip install -r requirements.txt`
3. Ensure you have a Groq API account and obtain an API key
4. Run the tool using `python app.py`

Note: Please replace `username` with your actual GitHub username.

## Tech Stack

* **Programming Language:** Python
* **Machine Learning Model:** LLaMA 3.1
* **Deep Learning Framework:** Groq API
* **Dependency Management:** pip
