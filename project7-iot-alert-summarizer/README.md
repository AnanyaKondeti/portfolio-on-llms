# 📡 Project 7: IoT Alert Summarizer

## What it does
Converts raw IoT sensor data into intelligent, 
prioritized alert summaries using LLM.

## Key Features
- Categorizes alerts into Critical, Warning, Normal
- Explains WHY each alert is important
- Suggests specific actions for each issue

## Connection to Real Work
Directly mirrors IoT monitoring work done at Texoham AI
where distributed IoT systems served 8,000+ daily users.

## Tech Used
- Python, Groq API, LLaMA 3.1
- Simulated IoT sensor data
