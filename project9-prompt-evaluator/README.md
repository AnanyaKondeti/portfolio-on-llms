# 📊 Project 9: Prompt Response Evaluator

## What it does
Automatically scores LLM responses across 5 quality criteria
using an LLM-as-judge approach.

## Evaluation Criteria
- Relevance: Does it answer the question? 
- Accuracy: Is information correct?
- Completeness: Is anything missing?
- Clarity: Easy to understand?
- Conciseness: Right length?

## Key Results
- Too short response: 29/50
- Good response: 43/50  
- Wrong answer: 28/50

## Why it matters
Companies use automated evaluation to test prompt quality
at scale without human reviewers.

## Tech Used
- Python, Groq API, LLaMA 3.1
